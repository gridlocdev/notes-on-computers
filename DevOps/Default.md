# Gridlocdev's notes on DevOps

DevOps is a philosophy of integrating the development, deployment, and maintenance of software between Development and Operations. This is accomplished through modern practices by automating much of the ceremony that divides code and the infrastructure it sits on in a more de-coupled and efficient way.
